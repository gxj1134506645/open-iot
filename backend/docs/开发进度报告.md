### Phase 5: 设备控制 ✅

#### 5.1 服务调用
- **ServiceCallRecord**: 服务调用记录（输入参数、输出参数、状态）

- **PropertySetRecord**: 属性设置记录（新旧值对比、设置类型）

#### 5.3 命令下发
- **CommandSendRecord**: 命令下发记录（命令参数、执行状态）

## 核心技术特性

### 1. 多租户隔离
- **TenantContext**: 租户上下文管理
- **数据隔离**: 所有表包含 `tenant_id` 字段
- **自动注入**: Gateway 自动注入 `X-Tenant-Id` 请求头

### 2. JSONB 字段支持
- **JsonbTypeHandler**: MyBatis Plus JSONB 类型处理器
- **用途**: 物模型定义（properties、events、services）、规则条件、告警上下文

### 3. Kafka 消息驱动
- **device-data 主题**: 设备数据上报（规则引擎监听）
- **device-control 主题**: 设备控制命令下发
- **异步处理**: 服务调用、属性设置、命令下发

### 4. 规则引擎
- **Aviator**: 表达式求值（`temperature > 50`、`humidity > 90 && temperature > 40`）
- **GraalJS**: JavaScript 脚本执行（复杂逻辑）
- **规则类型**: 设备规则、产品规则

### 5. 告警管理
- **告警级别**: critical、warning、info
- **状态流转**: pending → processing → resolved/ignored
- **批量处理**: 支持批量处理告警

### 6. 设备控制
- **服务调用**: 调用设备定义的服务
- **属性设置**: 设置设备属性（校验读写权限）
- **命令下发**: 下发自定义命令
- **异步执行**: 通过 Kafka 异步发送到设备
